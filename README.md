# 🤖 AI Resume Analyzer

A simple AI-powered tool that compares a resume with a job description and gives a match score, missing skills, and suggestions for improvement.

---

## 🛠️ Tech Stack

- FastAPI (Backend)
- Sentence Transformers (Hugging Face)
- scikit-learn (Cosine Similarity)
- PyPDF (Resume text extraction)

---

## ▶️ Usage

1. Upload a resume (PDF)
2. Enter a job description
3. Get:
   - Match score
   - Skills present
   - Skills missing
   - Recommendations
